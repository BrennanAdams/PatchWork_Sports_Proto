# Patchwork Sports Fitness App

## Getting Started

1. Clone the repository:
2. XCode on Mac OS to launch


## Features

- Personalized and sport-specific workouts
- Progress tracking
- Modern user interface with Dark Mode support
- In-app video tutorials

## Requirements

- Xcode 13 or later
- iOS 15 or later
- Swift 5 or later
